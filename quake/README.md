## Console

**Color Codes**

* `^0`: Black
* `^1`: Red 
* `^2`: Green
* `^3`: Yellow
* `^4`: Blue
* `^5`: Cyan
* `^6`: Purple
* `^7`: White
* `^8`: White (also?)
* `^9`: Gray
